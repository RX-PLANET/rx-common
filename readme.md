# FUYANLOVE

公共基础函数模块，路径常量，外部通用接口地址，全局主题等

## Step.1 Install

```
$ npm install @fuyanlove/pkg-common --save
```

## Step.2 Import

```javascript
import RxTime from "@fuyanlove/pkg-common/utils/rx-time.js";
```

## Js & Data Docs

- [utils](./docs/utils.md) 常用工具函数&过滤器等（获取落地页链接、图像处理等）
